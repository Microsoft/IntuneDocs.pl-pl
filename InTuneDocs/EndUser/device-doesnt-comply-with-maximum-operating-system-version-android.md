# Urządzenie nie jest zgodne z maksymalną wersją systemu operacyjnego

Urządzenie z systemem Android nie jest zgodne z zasadami administratora IT dotyczącymi maksymalnej wersji systemu operacyjnego. Aby uzyskać pomoc, skontaktuj się z administratorem IT.
